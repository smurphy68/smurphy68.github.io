Find My Hat

This coding challenge was a project inspired by my Codecademy learning. This was a node.js REPL project, however I have adapted it to work on a webpage. The tech stack was:

  • JavaScript

  • CSS3

  • HTML5

  • Node.js

Project Break Down

The aim of the game is to find the hat character. The user navigates through a series of randomly generated holes and field/tree characters. Wherever the player has been, a path character is replaced in the console. The player starts in the top left hand corner of a grid (if not taken up by the hat).

Hover over the grid with your mouse,
use the arrow keys to move your character!
If no move is possible, click below to refresh the board!
