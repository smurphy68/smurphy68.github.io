Tic Tac Toe

This was a coding challenge given to me by a friend, we are exchanging increasingly complicated games to code. The tech stack for this project was:

  • JavaScript

  • CSS3

  • HTML5

Project Break Down

An additional challenge for this project was to not use an OOP approach, which is generally how I have approached projects in the past.
The player clicks a box to make a move, filling it in with their character. The program then checks all possible win conditions to see if one has been fulfilled. If not, the player is toggled and the game continues. A square cannot be played over and the functionality is removed once the square is clicked.
If the board is full and there is no win condition, then the game is a draw. The user can then click the refresh button to play again!
